How did viewing a diff between two versions of a file help you see the bug that was introduced?

	It shows the changes that were made between the new version of the file and the old.  Can greatly reduce the amount of time to debug code if something goes wrong.

How could having easy access to the entire history of a file make you a more efficient programmer in the long term?
	
	You can see where possible errors occur and reference old code if something new breaks.
